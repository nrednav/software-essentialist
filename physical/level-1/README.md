# Node.js - Starter Kit

This is a starter kit for [Node.js](https://nodejs.org/en/) projects.

## Overview

This project has:

- Typescript
- ESLint
- Prettier
- Vitest
- Husky & lint-staged

## Setup

Install project dependencies:

```bash
npm install
```

Run app:

```bash
npm run dev
```

Run tests once:

```bash
npm run test
```

Run tests on-save:

```bash
npm run test:watch
```

Test coverage:

```bash
npm run coverage
```
